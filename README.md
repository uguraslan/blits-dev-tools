# Blits Development Tools

Official development tools for the [Lightning Blits](https://lightningjs.io/) framework.

## Packages

### [VSCode Extension](./vscode-extension)

[![VS Code Marketplace](https://img.shields.io/badge/VS%20Code%20Marketplace-lightning--blits-blue)](https://marketplace.visualstudio.com/items?itemName=lightningjs.lightning-blits)

Install from the VS Code Marketplace or search for **Lightning Blits** in the Extensions panel.

- Template syntax highlighting for `.blits` files and inline template strings
- Code completion for Blits components and attributes
- IntelliSense for Blits APIs

### [ESLint Plugin](./eslint-plugin-blits)

[![npm](https://img.shields.io/npm/v/@lightningjs/eslint-plugin-blits)](https://www.npmjs.com/package/@lightningjs/eslint-plugin-blits)

> _The first release is coming soon._

```bash
npm install --save-dev @lightningjs/eslint-plugin-blits
```

- Lint rules for Blits templates in `.blits` files and JS/TS template strings
- Catches common mistakes: missing keys on `:for` lists, invalid attribute values, structural errors

### [Prettier Plugin](./prettier-plugin-blits)

[![npm](https://img.shields.io/npm/v/@lightningjs/prettier-plugin-blits)](https://www.npmjs.com/package/@lightningjs/prettier-plugin-blits)

> _Still in active development — initial release in preparation._

```bash
npm install --save-dev @lightningjs/prettier-plugin-blits
```

- Consistent formatting for Blits templates in `.blits` files and JS/TS template strings
